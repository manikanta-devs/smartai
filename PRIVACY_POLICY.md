## 6. DATA SECURITY

### 6.1 Security Measures We Implement

🔒 **Encryption:**
- HTTPS/TLS for all data in transit
- AES-256 encryption for sensitive data at rest
- End-to-end encryption for passwords (bcrypt hashing)

🔒 **Access Control:**
- Role-based access control (RBAC)
- Only authorized employees access data
- Admin access logged and audited

🔒 **Authentication:**
- JWT token-based authentication
- Session timeouts after 30 minutes inactivity
- IP-based anomaly detection (future)

🔒 **Backups:**
- Daily encrypted backups
- Backup data stored separately
- Disaster recovery procedures tested quarterly

🔒 **Monitoring:**
- Real-time security monitoring
- Automated intrusion detection
- Security audit logs

### 6.2 What We Cannot Guarantee

⚠️ No system is 100% secure  
⚠️ Zero-day vulnerabilities exist  
⚠️ Social engineering attacks bypass tech  
⚠️ Employee theft (though we have safeguards)  

### 6.3 Your Responsibility

You must:
- Use a strong, unique password
- Never share your login credentials
- Use secure networks (not public WiFi)
- Enable 2FA when available (coming soon)
- Report suspicious activity immediately
